## FAQ

**Q: How much does RF HiRes Dash cost?**

Free tier is free forever. Pro tier coming soon.

**Q: Can I use RF HiRes Dash for commercial projects?**

Yes, both free and pro tiers support commercial use.

**Q: Is the source code available?**

Yes, on GitHub: https://github.com/your-org/rf-hires-dash

**Q: Can I run RF HiRes Dash on my own server?**

Yes, you can self-host. Instructions are in the GitHub repo.

**Q: How accurate are the predictions?**

RF HiRes Dash uses ITU-R P.368-9, the international standard. Accuracy depends on:
- Frequency (0.15–30 MHz only)
- Elevation data quality (SRTM ±16 m)
- Ground conductivity estimate (yours to provide)
- Terrain profile (included via SRTM)

Always verify with field measurements before deployment.

**Q: Can I export data for use in ArcGIS / QGIS?**

Yes, export as KMZ and import into any GIS software.
